Navigation & Menus
  • A: Select / Confirm
  • B: Back
  • X: Navigate (L1/R1 = skip 5)
  • Y: Search
  • Start / Select: Quick Access Menu
  • Hold A: View media, save states, or favourite
Hotkey Combos (R3)
  • R3 + D-Pad ↑↓: Volume
  • R3 + D-Pad ←→: Cycle Save Slots (0–999)
  • R3 + R1: Save Game
  • R3 + L1: Load Game
  • R3 + L2: Rewind (disabled on N64)
  • R3 + R2: Fast Forward
  • Start + Select (Hold): Quit Game
Arcade & RetroArch
  • Select (Arcade): Insert Coin
  • L3: Save / Load / Reset / Exit (Arcade only)
  • R3: Cabinet & Video Options (Arcade only)
  • L3 (Other Systems): Access RetroArch Menu
System Tips & SkyBit Secrets
  • Reboot from NAND: Boot to Android TV mode
  • RGB & CEC Fixes: Found in TV Box mode (disables unwanted auto-on)
  • Controllers: Hold HOME for 2–3 sec to power on (2x AAA batteries)
  • Auto-Off: Controllers enter sleep when inactive
